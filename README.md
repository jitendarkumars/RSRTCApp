# RSRTCApp
Rajasthan Hackathon Submission

To run the project : Requirements-
1. node.js
2. Android Studio/Xcode (with emulators)
3. react-native-cli
4. hasura-cli

Configurations- 
git clone https://github.com/jitendarkumars/RSRTC_APP
cd rsrtc_app
npm install 

Run the app -
react-native run-android/run-ios
